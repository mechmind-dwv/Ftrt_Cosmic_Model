# FTRT Cosmic Model

Proyecto de investigación helio-biológica y validación FTRT inspirado en Chizhevsky y la escala de la génesis armónica.  
Incluye:

- Modelos de fuerza de marea planetaria sobre el Sol
- Extracción de efemérides JPL y NOAA
- Comparación histórica de tormentas solares
- Predicción de eventos solares y sociales
- Scripts reproducibles y notebooks de análisis

## Estructura
- scripts/: Código Python principal
- data/: Datos históricos y de efemérides
- notebooks/: Exploración y visualización
- tests/: Pruebas de validación
