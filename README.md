# Ftrt_Cosmic_Model
 Proyeco de investigación helio-biológica y validación FTRT inspirado en Chizhevsky y la escala de la génesis armónica.  - Modelos de fuerza de marea planetaria sobre el Sol - Extracción de efemérides JPL y NOAA - Comparación histórica de tormentas solares - Predicción de eventos solares y sociales - Scripts reproducibles y notebooks de análisis
